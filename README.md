**Thematic vocabulary generator for fiction writers**

Writers don't need synonyms - they need *vibes*. 

A thesaurus tells you "walked" = "strolled". VibeWords tells you that a heist scene needs words like *infiltrate*, *adrenaline*, *vault*, *ticking clock*, and *chess game*.

## Why

Found this tweet: *"I wish there was a tool where you could input a theme and related words come out. Like input: heist, and every relevant word would come up - actions, objects, emotions, atmosphere."*

No good tool exists. Now it does.

## Run Locally

Just open `index.html` in your browser. No build step, no dependencies.

```bash
cd playground/vibewords
open index.html  # macOS
xdg-open index.html  # Linux
```

Or use a local server:
```bash
python3 -m http.server 8080
# Open http://localhost:8080
```
